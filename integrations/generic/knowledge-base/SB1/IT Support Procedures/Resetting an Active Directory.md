# Resetting an Active Directory Account

## Steps

1. **Verify request authenticity:**  
   Confirm user identity via callback or MFA.
2. **Disable the account temporarily.**
3. **Reset the password:**  
   Use a strong, randomly generated password.
4. **Re-enable the account.**
5. **Notify the user securely** (do not send the new password by email).
6. **Document the reset** in the ticketing system and TheHive case if IR-related.
