# Security Incident Response – Glossary

| Term  | Definition |
|-------|------------|
| IOC   | Indicator of Compromise. Any data point (file, hash, IP, domain, etc.) that suggests a security breach. |
| TTP   | Tactics, Techniques & Procedures. The behavior or modus operandi of a threat actor. |
| TLP   | Traffic Light Protocol. Information sharing marking (e.g., TLP:RED, TLP:AMBER, TLP:GREEN, TLP:CLEAR). |
| SOAR  | Security Orchestration, Automation, and Response. A stack of tools to automate and coordinate security processes. |
| MITRE ATT&CK | A knowledge base of adversary tactics and techniques based on real-world observations. |
| IR    | Incident Response. The process of managing and mitigating cybersecurity incidents. |
| PAP   | Permissible Actions Protocol. Guidance on what recipients can do with the information. |
| False Positive | An alert or event that, after investigation, turns out to be benign or not a threat. |
| Containment | Actions taken to limit the spread or impact of a security incident. |
| EDR   | Endpoint Detection and Response. Security tools focused on monitoring and responding to endpoint threats. |

**Tip:**  
Keep this glossary updated as new terms are adopted in your processes.
