# Acceptable Use & Data Handling – SOC Quick Reference

- **No personal USB drives** on any SOC workstation
- **Sensitive data:** Only share via encrypted email or approved platform
- **Remote work:** Use only managed devices with full disk encryption, EDR/XDR, and up-to-date patches
- **All logs must be retained for 365 days (per policy section 5.3)**
