# Known Issues & Quick Fixes – SOC Tools

## VPN Login Error
- **Symptom:** Repeated disconnects when authenticating from overseas.
- **Solution:** Open ticket with IT, reference “VPN error workaround doc”.

## Email Gateway: Blocklist Sync Delay
- **Issue:** Malicious domains not instantly blocked after IOC update.
- **Note:** Expect 30-minute propagation time. If urgent, escalate to Email Admin (see contacts).
